# Everything Is Pixel — Roadmap complète (Web‑first, modulaire)

## 1️⃣ Vision générale
Idle game 2D en pixel‑art où **chaque élément visible peut être réduit en pixels**, recyclé, puis valorisé pour créer d’autres objets ou effets.
Boucle de base : **Générateur → Pixel → Transformeur → Résultat → Valoriseur → XP / Argent / Effets.**
